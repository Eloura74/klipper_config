
Voici les étapes à suivre pour avoir votre propre theme dans Mainsail:

- Je passe par kiauh pour avoir un theme de base que l'on pourras modifier par la suite:
  - Ouvrer kiauh en lancant la commande " ./kiauh/kiauh.sh ".
  - Aller sur " Advance " en tappant " 4 ".
  - Puis " Theme Installer " en tappant " 6 ".
  - Choisir ensuite un thème à installer.

- Une fois le thème installé un fichier .theme sera crée dans votre configuration.
Voici le détail de celui-ci:
  -custom.css : permet d’intégrer du code css sur des éléments
  -favicon-32×32.png : icone dans la barre du navigateur
  -main-background.png : le fond de l’écran principal
  -sidebar-background.png : Bande gauche de l’écran avec le menu
  -sidebar-logo.svg : logo dans la barre de titre
