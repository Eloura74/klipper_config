#     - Le fichier " Moteur Cam " avec à l'intérieur:

#             - La section Manual_stepper : Avec toute la configuration des pins.
#             - La section TMC2209 : Avec toute la configuration des TMC.
#             - La Macro HAUTEUR CAMERA : Qui nous permet de monter ou descendre la caméra.
#             - La Macro CAM ON : Qui active le moteur de la caméra.
#             - La Macro CAM OFF : Qui désactive le moteur de la caméra.
#             - La Macro Cam Home : Qui permet de faire un Home de l'axe de la caméra.
